Teacher's Pet

The Favorite Student

Teacher's Pet is an newly designed, innovative grade-book which holds some ground-breaking features that decreases the gruesome process of using an out-of-date, and non-user friendly online grade-book.

These features include: the ability to curve entire grade-sets using one of our pre-calacuted curve buttons, and to view the impact each curve has, in real time, using one of graph views. Another feature includes the ability to email a progress report of each student with just a click of a button!
Tech: The Back-end was bulit using Java with Spring, using a PostgreSQL Database. The Front-end was built using AngularJS and HTML5 and CSS3. The APIs Chart.js, Angular-Chart.js and SendGrid were also used
